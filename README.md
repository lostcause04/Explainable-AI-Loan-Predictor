# 💰 Loan Approval Predictor (Explainable AI)

This project explores whether Explainable AI (XAI) is necessary or overrated using a real-world case study: loan approval prediction.

## 🚀 Features
- Predict loan approval using Machine Learning
- Interactive web app using Streamlit
- Explainability using feature importance

## 🧠 Models Used
- Random Forest (Black-box model)
- Decision Tree (Explainable model)

## 📊 Explainability
- Feature importance visualization
- Insights into decision-making

## 🖥️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
